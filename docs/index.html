CODA jQuery Plugin

Introduction

This plugin is written to help with the integration of various reporting functionality tied to CODA. Specifically:

  • Link Events
  • Search related reporting functions
We can extend this to do photo specific reporting, and other things further down the road.

Link Events

Code investigates the link. It will do a link event call "e" (external) if an attribute rel="external" is present in the anchor tag. Otherwise it will use an "o" (internal)

Search Related Events
